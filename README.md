## 기능 구현

✔️ 지역 시간 표기법에 따른 타이머 출력

✔️ 입력값에 따른 오름차순, 내림차순 정렬

✔️ 오름차순 정렬 후 3초 뒤 내림차순 결과 출력

✔️ 숫자와 콤마를 제외한 데이터 입력 시 예외처리

✔️ 데이터 정렬 후 입력 데이터 초기화

</br>

## 설치 및 시작

```
npm i
npm start
```
